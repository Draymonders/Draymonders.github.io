- 虚拟内存是什么用途，如何实现？
    * 首先需要知道物理地址和逻辑地址
    * 虚拟内存使得应用程序认为它拥有连续的可用的内存（一个连续完整的地址空间），而实际上，它通常是被分隔成多个物理内存碎片，还有部分暂时存储在外部磁盘存储器上，在需要时进行数据交换。
- 进程的通信方式有哪些
    * 匿名管道(亲缘关系)
    * 具名管道
    * 消息队列
    * 信号量
    * 信号
    * 共享内存
    * socket
- 页面置换算法
    * 最佳(所选择的被换出的页面将是最长时间内不再被访问)
    * 最近未使用(LRU)
    * 最少频率未使用(LFU)
    * 先进先出
    * 时钟
- 分页和分段有什么区别
    * 分页大小相同 页表
    * 分段大小不同 段表
    * 分页主要用于实现虚拟内存，从而获得更大的地址空间；分段主要是为了使程序和数据可以被划分为逻辑上独立的地址空间并且有助于共享和保护
