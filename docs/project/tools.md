## 二进制文件内容查看

Sublime

- 快捷键`Command+Shift+P`，选择`Package Control`: 
- Install Package安装插件搜索`HexViewer`并等待安装成功
- 打开要查看的二进制文件，这个时候还是没有以HexViewer模式打开的按下快捷键`Command+Shift+P`，搜索并选择`HexViewer`
- `Toggle Hex View`，回车骚等片刻后即可
